## 🚀 What I'm Building

### 🛡 [Secret Sanitizer](https://github.com/souvikghosh957/secret-sanitizer-extension)
A Chrome extension that detects and masks secrets — API keys, passwords, tokens, and more — before they leave your machine. Fully local, zero telemetry, open source. Because your credentials shouldn't end up in someone else's training data.

### 🦉 [SurpriseSage](https://github.com/souvikghosh957/SurpriseSage)
A local-first AI companion that lives in your Mac's menu bar. Delivers context-aware wisdom, motivation, and micro-surprises tailored to your goals and mood. Powered by Ollama or cloud LLMs. Privacy-first, with RAG memory that never repeats itself.

### 🔍 [Anomalyze Engine](https://github.com/souvikghosh957/anomalyze-engine)
Real-time AI-powered anomaly detection for network traffic. Built for speed, accuracy, and low false positives — because alert fatigue is a real problem.
